# Weather Forecast Application

The Weather Forecast Application is a responsive web application that provides current weather conditions and 5-day forecasts for any specified city using the OpenWeather API. Users enter a city name to retrieve detailed weather information, including humidity, wind speed, and other metrics. The app features a toggle button for seamlessly converting temperature displays between Celsius and Fahrenheit.

## Features

- Search weather by city name
- Get weather for current location
- Toggle between Celsius and Fahrenheit
- 5-day weather forecast
- Weather alerts for extreme temperatures
- Mobile-friendly design

## How to Run

1. Clone or download this repository - https://github.com/dilnakt-bnglr/weather-forecast-app
2. Open `index.html` in your browser.
3. Enter a city name or use the current location button.

## File Structure

- `index.html` — Main HTML file
- `index.js` — JavaScript logic (API calls, UI updates, toggle functionality)
- `output.css` — Compiled CSS (Tailwind + custom styles)
- `input.css` — Source CSS (if using Tailwind)

## API

- Uses [OpenWeatherMap API](https://openweathermap.org/api)
- Replace the `api_key` variable in `index.js` with your own API key if needed
